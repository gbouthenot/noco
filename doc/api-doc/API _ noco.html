        noco_account

        Parameters

        Parameter Value Description Parameter Type Data Type
        id_show Show ID path integer
        body
        		Progress informations body
        • Model
        • Model Schema
        Progress_infos {
        progress (Progress),
        resume_play (integer, optional): Next time, start this show at this TC (in ms)
        }
        Progress {
        total (integer): Total percentage of the show viewed,
        detail (array[Progress_detail], optional)
        }
        Progress_detail {
        tc_start (integer): In milliseconds, start of this part,
        tc_end (integer): In milliseconds, end of this part
        } 
        {
  "progress": {
    "total": 0,
    "detail": [
      {
        "tc_start": 0,
        "tc_end": 0
      }
    ]
